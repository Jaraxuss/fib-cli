fib-cli
===
CLI for [fib-app](https://github.com/fibjs/fib-app/)

## Install
```
npm install fib-cli -g
```

## API

1. create a basic project based on fib-app
    ```
    fib-cli --init
    ```
    or
    ```
    fib-cli -i
    ```

2. create a model based on fib-orm
    ```
    fib-cli --newmodel
    ```
    or
    ```
    fib-cli -n
    ```
